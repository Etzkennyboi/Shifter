Start login flow ΓÇö sends OTP to email, or AK login if no email provided

Usage: onchainos.exe wallet login [OPTIONS] [EMAIL]

Arguments:
  [EMAIL]  Email address to receive OTP (optional ΓÇö omit for AK login)

Options:
      --base-url <BASE_URL>  Backend service URL (overrides config)
      --locale <LOCALE>      Locale (e.g. "en-US", "zh-CN"). Optional
      --chain <CHAIN>        Chain: ethereum, solana, base, bsc, polygon, arbitrum, sui, etc
      --force                Force re-login, skip API Key switch confirmation
  -h, --help                 Print help
