Agentic wallet: login, verify, create, switch, status, logout, balance

Usage: onchainos.exe wallet [OPTIONS] <COMMAND>

Commands:
  login          Start login flow ΓÇö sends OTP to email, or AK login if no email provided
  verify         Verify OTP code from email
  add            Add a new wallet account
  switch         Switch active account
  status         Show current wallet status
  addresses      Show wallet addresses grouped by chain category (XLayer, EVM, Solana)
  logout         Logout and clear all stored credentials
  chains         List all supported chains (cached locally, refreshes every 10 minutes)
  balance        Query wallet balances
  send           Send a transaction (native or token transfer)
  history        Query transaction history or detail
  contract-call  Call a smart contract (EVM inputData or SOL unsigned tx)
  help           Print this message or the help of the given subcommand(s)

Options:
      --base-url <BASE_URL>  Backend service URL (overrides config)
      --chain <CHAIN>        Chain: ethereum, solana, base, bsc, polygon, arbitrum, sui, etc
  -h, --help                 Print help
